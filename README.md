# 3D Box Modeler

カスタム収納ボックスをブラウザ上で簡単にデザインして、3Dプリント用のSTLファイルをダウンロードできるツールです。

## 特徴

- **ブラウザだけで完結**: インストール不要、HTMLファイルを開くだけで使える
- **リアルタイムプレビュー**: パラメータを変更すると即座に3Dモデルが更新
- **精密な寸法設定**: スライダーと数値入力の両方に対応
- **仕切り機能**: 縦横の仕切りを自由に追加
- **STLエクスポート**: 3Dプリント用のSTLファイルを直接ダウンロード

## 使い方

### 基本操作

1. **HTMLファイルを開く**: `box-modeler-react.html`をブラウザで開く
2. **パラメータを調整**: 左側のコントロールパネルで寸法を設定
3. **3Dビューで確認**: マウスで回転・ズーム
   - **ドラッグ**: 回転
   - **ホイール**: ズーム
4. **STLダウンロード**: "Download STL"ボタンでファイルを保存

### パラメータ説明

#### Box Parameters（箱の寸法）
- **Width**: 箱の幅（20-300mm）
- **Depth**: 箱の奥行き（20-300mm）
- **Height**: 箱の高さ（10-200mm）
- **Wall Thickness**: 壁の厚さ（1-10mm）
- **Bottom Height**: 底板の高さ（0.5-20mm）
  - 高くすると内部容積が減る
  - 底上げ収納に便利

#### オプション
- **Enable Fillet (1mm)**: 角を1mm丸める（面取り）
  - プリント後の手触りが良くなる
  - 注意: 2方向のみ面取りされます
- **Extend Partitions to Bottom**: 仕切りを地面まで伸ばす
  - OFF: 底板の上で止まる（スタッキング向き）
  - ON: 地面まで貫通（底まで完全分割）

#### Partitions（仕切り）
- **Vertical Partitions**: 縦仕切りの数（0-10本）
- **Horizontal Partitions**: 横仕切りの数（0-10本）
- 仕切りは常に壁より1mm低く設定されます
- 各セルは自動的に均等に配置されます

#### Cell Dimensions（セルサイズ表示）
- **Internal Height**: 内部の高さ（底板から壁の上端まで）
- **Cell Width**: 各セルの幅
- **Cell Depth**: 各セルの奥行き

すべてmm単位で表示され、仕切りの厚みも考慮した実寸です。

## 3Dプリント時の注意

### スライサー設定
- **向き**: STLは自動的に正しい向き（底面が下）で出力されます
- **サポート**: 通常は不要
- **インフィル**: 20-30%推奨
- **壁の厚さ**: 2-3ライン推奨

### 推奨設定例
```
寸法: 100×100×50mm
壁厚: 2mm
底高: 2mm
仕切り: 2×2（9セル）
→ 各セル: 約30×30mm
```

## 技術仕様

- **座標系**: Z軸上向き（スライサーと同じ）
- **単位**: ミリメートル（mm）
- **ファイル形式**: ASCII STL
- **ブラウザ要件**: モダンブラウザ（Chrome, Firefox, Safari, Edge）

## ファイル共有・公開

### 友人に使ってもらう
1. **ファイルを直接送る**
   - `box-modeler-react.html`をメール/Slack/Dropboxで送信
   - 相手はブラウザで開くだけで使える
   - インターネット接続が必要（CDNからライブラリを読み込むため）

### ウェブで公開する

#### GitHub Pages（無料・簡単）
1. GitHubにリポジトリを作成
2. `box-modeler-react.html`を`index.html`にリネーム
3. Settings → Pages で公開
4. `https://yourusername.github.io/リポジトリ名/`でアクセス可能

#### Netlify / Vercel（無料・即座に公開）
1. サイトにアクセス
2. HTMLファイルをドラッグ&ドロップ
3. 即座にURLが発行される

## トラブルシューティング

### 3Dビューが表示されない
- ブラウザのJavaScriptが有効か確認
- コンソールにエラーが出ていないか確認（F12で開発者ツール）

### STLファイルが開けない
- スライサーソフトが最新版か確認
- ファイルサイズが0でないか確認

### 仕切りの配置がおかしい
- セルは均等に配置されます
- Cell Dimensionsで実寸を確認してください

### プリント時にサイズが合わない
- スライサーの単位設定を確認（mm）
- インポート時に自動スケーリングがOFFか確認
