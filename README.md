
# 3D Box Modeler（3Dボックス・モデラー）

ブラウザだけで動作する、カスタム収納ボックスを設計し、  
3Dプリント用の STL ファイルを生成できる Web アプリです。

👉 **[アプリを開く](https://daijiro22.github.io/box-modeler/)**

![Screenshot](screenshot.png)

## ✨ 主な特徴

- 🌐 **インストール不要の完全ブラウザアプリ**
- ⚡ **リアルタイム3Dプレビュー**
- 📏 **正確な寸法設定（スライダー＋数値入力）**
- 📦 **仕切りを自由配置（縦・横に最大10本）**
- 🧮 **セル寸法を自動計算**
- 💾 **STLファイル書き出し対応**
- 🎨 **フィレット（角丸）オプション付き**

## 📖 基本操作

* **ドラッグ**：回転
* **ホイール**：ズーム

---

## 📏 設定できるパラメータ

### ボックスの外寸

| 項目                 | 範囲        |
| ------------------ | --------- |
| 幅（Width）           | 20–300 mm |
| 奥行き（Depth）         | 20–300 mm |
| 高さ（Height）         | 10–200 mm |
| 壁厚（Wall Thickness） | 1–10 mm   |
| 底上げ（Bottom Height） | 0.5–20 mm |

### 仕切り（Partitions）

* 縦方向：0〜10
* 横方向：0〜10
* 自動的に均等配置
* 仕切りは壁より1mm低く積み重ねしやすい設計

### オプション

* フィレット（角丸）：0〜10 mm
* 底板位置の可視化（点線）

---

## 🧰 3Dプリントのコツ

* **向き**：底面を下向き
* **サポート**：不要
* **インフィル**：20–30%
* **壁ライン**：2〜3本

### 例：

```
外寸：100 × 100 × 50 mm  
壁厚：2 mm  
底上げ：2 mm  
仕切り：2 × 2（全9セル）  
セルサイズ：約30 mm 四方  
```

## 🐛 トラブルシューティング

**3Dビューが表示されない場合**

* ブラウザのコンソール(F12)をチェック
* JavaScriptが有効か確認
* 別ブラウザで試す

**STLが読み込めない場合**

* ファイルサイズが0でないか確認
* スライサーの単位をmmに設定
* 読み込み時の自動スケールをOFF
